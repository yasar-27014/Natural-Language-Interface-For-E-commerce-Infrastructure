# Natural-Language-Interface-For-E-commerce-Infrastructure
ShopEase — E-Commerce Chatbot
What it is:
A chatbot built using Python that answers customer queries for an e-commerce store. You type a question in plain English — it matches your keywords and replies instantly. It covers orders, payments, returns, shipping, offers, and support.
Two versions:

Terminal version — runs in the black command window
Streamlit version — runs as a proper website in your browser

How it works in one line:

Your input → converted to lowercase → checked against keyword list → matching reply is shown


Commands
To open the Streamlit website chatbot:
bashpython -m streamlit run app.py
To open the terminal chatbot:
bashpython chatbot.py
